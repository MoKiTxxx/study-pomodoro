# UI 改善方法：以「工作階段狀態」為核心

這個專案最適合的 UI 改善方向，不是增加更多裝飾，而是讓使用者在不同狀態下只看到當下需要的操作。番茄鐘工具的核心情境很明確：設定、專注、休息、完成、回顧。UI 應該圍繞這些狀態切換，而不是把所有功能同時放在畫面上。

## 改善目標

- 設定時專注填資料，不顯示倒數面板。
- 計時時專注看時間，不顯示設定表單。
- 休息時清楚顯示目前是休息，不讓使用者誤以為專注仍在進行。
- 完成後清楚顯示結果，並提供「開始新工作階段」。
- 今日紀錄與累積統計保持資料密度，但減少不必要的文字干擾。

## 核心方法

採用「狀態導向 UI」。

也就是根據 timer 狀態決定畫面：

```text
idle          -> 顯示開始設定表單
focus         -> 顯示計時面板
break         -> 顯示計時面板，狀態用綠色標示休息中
paused        -> 顯示計時面板與 Resume
completed     -> 顯示結果與開始新工作階段
stopped       -> 顯示已停止與開始新工作階段
```

這樣可以避免同一頁同時出現「開始讀書」和「目前計時」，降低誤操作。

## 畫面結構建議

### 1. 開始讀書頁

設定畫面只保留開始前需要輸入的內容：

- 任務類型
- 科目
- 書本或課程
- 章節
- 計時模式
- 時間設定
- 開始前備註
- Start

改善方式：

- 任務類型放在最上方。
- 科目與書本或課程並排。
- 手動模式與自動模式用 segmented/radio 控制。
- 自動模式只顯示：
  - 總時間
  - 幾段
  - 總休息時間
- 換算結果分行顯示，不用塞在同一行。

### 2. 計時頁

計時畫面只保留當下需要看的資訊：

- 狀態
- 剩餘時間
- 已專注時間
- 番茄鐘進度
- 休息時間
- Pause
- Resume
- Stop work

改善方式：

- 「專注中」與「休息中」使用明顯狀態色。
- 剩餘時間可以隱藏，避免造成壓力。
- 按鈕固定三個：Pause、Resume、Stop work。
- 不再顯示「手動完成」或「儲存部分紀錄」這類語意不清的操作。

### 3. 完成頁

完成後要讓使用者知道這次結果，不要直接回到設定表單。

建議顯示：

- 本次狀態
- 已專注時間
- 完成番茄鐘數
- 休息時間
- 同步 Google Sheets 狀態
- 開始新工作階段

### 4. 今日紀錄

今日紀錄應該偏向「快速檢查與補資料」。

改善方式：

- 上方保留三個摘要數字：
  - 今日專注時間
  - 今日讀書時數
  - 今日番茄鐘數
- 表格保留完整資料，但隱藏不常看的欄位。
- 編輯區只保留：
  - output
  - stuck
  - next_action

### 5. 累積統計

累積統計應該偏向「看趨勢」。

改善方式：

- 第一列顯示總時數、總番茄鐘、7 日平均、30 日平均。
- 第二區顯示各科與任務類型統計。
- 第三區顯示每日讀書時數折線圖。
- 每週表格放在圖表下方。

## 實作原則

1. 不新增太多文字說明  
   UI 應該靠排列、狀態、按鈕名稱讓使用者理解，不要在畫面上放大量教學文字。

2. 不把卡片套卡片  
   Streamlit 畫面容易變擠，頁面區塊應該用標題、columns、spacing 區分。

3. 按鈕名稱要表示動作  
   例如 `Stop work` 比 `Stop` 清楚，`Start new work session` 比 `New session` 清楚。

4. 資料寫入值保持英文  
   UI 可以中文或英文，但 Google Sheets 裡的 status、task_type 等系統值要保持英文，方便統計與搜尋。

5. 把計算邏輯和 UI 分開  
   時間換算、timer state、Google Sheets 寫入不要塞在 UI 控制裡，避免後續改畫面時改壞資料邏輯。

## 建議實作順序

1. 整理開始頁表單
2. 整理計時頁資訊階層
3. 統一按鈕名稱與狀態顏色
4. 優化今日紀錄表格欄位
5. 優化統計頁排版
6. 最後再調整細節樣式

不要一開始就改顏色和裝飾。先把流程和狀態整理清楚，視覺才會穩。

## 驗收標準

完成 UI 改善後，應該符合：

- idle 狀態只看到設定表單。
- focus / break / paused 狀態只看到計時面板。
- completed / stopped 狀態停留在計時結果畫面。
- 使用者不需要猜哪個按鈕會寫入資料。
- Google Sheets 寫入格式不因語言切換而改變。
- 手機與桌面上文字不重疊。
- Alarm 播放不會讓計時停止刷新。
